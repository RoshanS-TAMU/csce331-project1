WannaCry Malware Analysis & Defense Another placeholder link because I haven't hosted this one either

WannaCry was one of the most famous and most destructive ransomware attacks in modern history. Believed to originate from the North Korean government, the virus once downloaded will encrypt all files on its host machine, and then generate copies of itself throughout the host network, infecting and destroying more and more devices. In this writeup, I documented a walkthrough of the virus features, from static analysis using tools like CFF Explorer and Ghidra, to dynamic analysis including debugging, and finally running the virus and using a killswitch found in the virus's decompiled code to recover my files from the machine.
